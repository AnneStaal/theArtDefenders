Backstory

Welke kunstschat heb ik, Rijkje, beschermd?

Elke transportkist had een code, zodat je kon zien welke kunstschat in de kist zat. De kunstschatten in de kisten verschilden heel erg. Het kan een schilderij zijn, een stuk van een Engelse vloot, een stoel, een bokaal en nog veel meer. Elke Rijkswachter heeft op zijn rug een code staan. Via de website kan je zien welke kunstschat jouw Rijkswachter heeft beschermd.

Zelf heb ik, Rijkje, ook een code op mijn rug. Ik heb namelijk ’Bokaal met getordeerde ribben beschermd’ tijdens de verbouwing. Deze Bokaal is gemaakt in Venetië rond 1875.
